# Email Classification System

## Overview
This project classifies emails into categories such as spam and not spam using Natural Language Processing (NLP) techniques.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLTK
- TF-IDF Vectorizer

## Problem Statement
Email spam is a major issue. The goal is to automatically classify emails using machine learning.

## Approach
- Text preprocessing (cleaning, tokenization)
- Feature extraction using TF-IDF
- Model training (Naive Bayes / Logistic Regression)
- Model evaluation

## Results
- High accuracy in spam detection
- Efficient text classification pipeline
